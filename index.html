O que é o projeto "Zenplace"?

O projeto "Zenplace" foi desenvolvido por alunos do primeiro período do curso de Sistemas para Internet na UNIESP. A UNIESP proporciona certos Hackathons, que, nós estudantes decidimos participar para finalidades educacionais.

Zenplace:

Zenplace é um ambiente virtual que se utiliza do ChatGPT para o usuário desabafar um pouco dos seus problemas enquanto se acalma escutando músicas relaxantes ao seu gosto. O projeto não visa substituir o estudo e trabalho de um Psicólogo ou Psiquiatra, apenas queremos incentivar a sua importância e que o usuário procure o tratamento adequado.
